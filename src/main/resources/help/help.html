О программе

Программа CloudSync предназанчена для синхронизации локальной файловой системы с различными облачными хранилищами данных.

Основные возможности программы.

1. Подключение учетных записей облачных хранилищ от различных компаний.
2. Просмотр содержимого каталогов хранилищ данных.
3. Перемещение по каталогам различных хранилищ данных.
4. Сравнение каталогов в отдельном потоке выполнения программы.
5. Выбор версий файлов при синхронизации каталогов.
6. Выбор направления синхронизации (из локального хранилища в облачное, из облачного хранилища в локальное, в обе стороны).
7. Синхронизация каталогов в отдельном потоке выполнения программы.
8. Создание задачи по синхронизации каталогов в заданное пользователем время.
