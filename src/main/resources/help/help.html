О программе

Программа CloudSync предназанчена для минимизации передаваемой информации при синхронизации хранилищ данных.

Основные возможности программы.

1. Подключение учетных записей облачных хранилищ от различных компаний.
2. Подключение к удаленному серверу с данными по IP адресу и номеру порта сервера.
3. Просмотр содержимого каталогов хранилищ данных.
4. Перемещение по каталогам различных хранилищ данных.
5. Сравнение каталогов в отдельном потоке выполнения программы.
6. Выбор версий файлов при синхронизации каталогов.
7. Выбор направления синхронизации (из локального хранилища в удаленное, из удаленного хранилища в локальное, в обе стороны).
8. Синхронизация каталогов в отдельном потоке выполнения программы.
9. Создание задачи по синхронизации каталогов в заданное пользователем время.
